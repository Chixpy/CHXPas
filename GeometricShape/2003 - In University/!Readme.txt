This is a practical exercise that I did at university for a Programming III class in 2003. Yes, we used Turbo Pascal and Delphi 6; and we had to study the patterns from "Design Patterns: Elements of Reusable Object-oriented Software" by Gang of Four (and the book in English even though the exam was in Spanish).

For other classes we had to learn C, C++ and Java on our own.

It's all in Spanish but you can see the style rules that I used at that time. There were no specific rules except having to indicate the preconditions, postconditions, and a description of each method.

Sí, usábamos Turbo Pascal y Delphi 6; y teníamos que estudiar los patrones de "Design Patterns: Elements of Reusable Object-oriented Software" por Gang of Four (y el libro en inglés a pesar de que el examen era en español).

Para otras clases teníamos que aprender C, C++ y Java por nuestra cuenta.

Está todo en español pero se pueden ver las normas de estilo que usaba yo por aquel entonces. No había reglas específicas excepto el tener que indicar las precondiciones, poscondiciones y una descripción de cada método.
